# How to use CSS

1. CSS stylesheet을 따로 만들어 html `link:css` + `tab` 사용해 import
2. html 파일의 `<style>` 태그 안에 스타일을 작성
3. in-line style: 요소에 직접 style을 박는 방법 ⇒ 유지보수가 어려움
