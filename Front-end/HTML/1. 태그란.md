# 태그 Tag

```html
 <p> 내용 </p>
```

마크업할 내용을 감싸면서 content가 어떤 성격을 가지는지 알려줌

---

# Tag의 구성 요소 attribute

### 속성 attribute을 통해 추가적인 정보를 제공한다.

```html
 <p attr="value"> </p>
```

Tag만으로는 담지 못 하는 추가적인 정보를 제공한다.

<p>태그에 추가적인 정보를 주는 language attribute

Ex.  
```html
<p lang="ko"> 안녕하세요 </p>
```

어떤 Tag들은 필수적으로 attr를 줘야 한다!
