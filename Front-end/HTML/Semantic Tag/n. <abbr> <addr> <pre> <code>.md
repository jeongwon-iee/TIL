# abbreviation 약자, 약어


```html
<abbr title=""> </abbr>
```

⇒ 마우스를 갖다 대면 title에 적은 내용이 나옴

# preformatted text

브라우저는 엔터키를 눌러도 스페이스로 처리해버림

이 때 `<p>` 태그 대신 `<pre>` 태그를 사용하면 입력한 그대로 브라우저에 나타남(indent도 반영)

---

# code

보통 코드를 작성하고 싶을 때 <pre> 안에 <code>를 사용하는 경우가 많음
