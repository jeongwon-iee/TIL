# Semantic Markup

의미 구조에 맞게 html을 잘 작성하는 것 !

⇒ 브라우저가 내가 전달하고자 하는 컨텐츠의 중요도를 잘 파악

⇒ 다양한 html tag를 이용해서 그 정보 컨텐츠의 의미에 맞게 구조적으로 코드를 짜면 내가, 다른 사람이 보기에도 편한 코드가 될 수 있음. div를 쳐발쳐발X 함께 일 하기 좋은 동료가 된다.

⇒ 검색 엔진 최적화에 도움이 됨.

브라우저는 html에 Markup한 대로 중요도를 파악
